# Reimbursement Management System (RMS) VJTI

A platform for the institutional people to apply for reimbursements for any activity/achievement and keeping audit for the approval flow from student/professor to HOD to accounts department with all the statistics.

## 📊 UI Dashboard Features

- **Departmental segregation**
- **Approval/Rejection flow with real time updates**
- **Achievements management**
- **Role based access**
- **Email authentication**
