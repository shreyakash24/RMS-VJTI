# Reimbursement Management System (RMS) VJTI

A platform for the vjti people to apply for reimbursements for any activity/achievement and keeping audit for the approval flow from student/professor to HOD to accounts department with all the statistics.

## 📊 Features

- **Departmental segregation**
- **Approval/Rejection flow with real time updates**
- **Achievements management**
- **Role based access**
- **Email authentication**

## Tech Stack

- Frontend: react.js, vite
- Backend: node.js, express.js
- Database: PostgreSQL
